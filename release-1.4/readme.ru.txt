== Description ==

Trumpcha is a very unique jQuery ajaxed CAPTCHA plugin that requires just one click/(mouse move) instead of typing characters, calculate or recognize weak-distinguishable pictures.

Trumpcha это уникальная jQuery ajax CAPTCHA система, предназначенная для защиты от спам-ботов. 
Пользователю достаточно одного клика или движения мышью, не надо ничего печатать, подсчитывать, вглядываться в еле различимые картинки.

[Plugin Downloads]: (http://cane.0fees.net/trumpcha/)


== Installation ==

1. Upload `wp-trumpcha` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. That's all!



== Frequently Asked Questions ==

= How does Trumpcha work? =

Feature of work Trumpch'a consists in the following. There are 2 boxings - the fleet and the house. In fleet there are 4 playing cards, on one on each suit. 
In a random way trump suit (one of four presented) gets out. It is necessary to transfer a card with trump suit to the house.
To throw off a trump in the house it is possible in two ways:
1. We hook the mouse on trump suit and move it in the house.
2. Mouse click on a special icon on trump suit. As a result suit itself will get to the house.
If suit not trump, it does not get to the house.
Only after trump hit in the house, the user can send the comment.
The warning message will be otherwise given out.


Особенность работы Trumpch'a заключается в следующем. Имеется 2 бокса - флот и дом. Во флоте находятся 4 карты, по одной на каждую масть.
Случайным образом выбирается козырная масть (одна из четырех представленных). Необходимо перенести карту с козырной мастью из флота в дом.
Сбросить козырь в дом можно двумя способами: 
1. зацепив мышью козырную карту, переместить ее в дом.
2. нажав на специальной иконке, на казырной карте. далее она сама переместиться в дом. 
Не козырные масти в дом попасть не могут.
При попадании козыря в дом, пользователь может отправить свой комментарий. В иных случаях будет выдано предупреждающее сообщение.
